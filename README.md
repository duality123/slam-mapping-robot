<h1>SLAM MAPPING ROBOT</h1>



<p>This project is a spider-style quadruped robot designed for mapping tasks. It uses an ESP32-CAM as the “head” and an Arduino Nano as the main controller. The robot builds maps by detecting collisions and distance using an HC-SR04 ultrasonic sensor. A web-based application is included so users can visualize the generated map and monitor the robot’s surroundings in real time.</p>
